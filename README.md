# 📝 Todo Dashboard Web App

A modern full-stack Todo Dashboard with authentication, task tracking, and profile management.

---

## ✨ Features

- 🔐 User Authentication (Signup / Login)
- 👤 Profile with Image Upload (Multer)
- 📋 Add, Delete & Complete Tasks
- 📅 Due Date & Priority System
- 📊 Progress Bar Tracking
- 👥 View All Users
- 🎨 Modern Dark UI
- ⚡ Fast & Interactive Experience

---

## 🛠️ Tech Stack

- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript (Vanilla JS)
- 🟢 Node.js
- 🚀 Express.js
- 🍃 MongoDB (Mongoose)
- 📦 Multer (File Upload)

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository
git clone https://github.com/Shraddha-1803/Todo-Dashboard-WebApp.git
cd Todo-Dashboard-WebApp


### 2️⃣ Install dependencies
npm install


### 3️⃣ Create .env file (inside backend)
MONGO_URI=mongodb://127.0.0.1:27017/todoDB
PORT=5000


### 4️⃣ Run backend
node backend/server.js


### 5️⃣ Run frontend
Open: frontend/index.html
